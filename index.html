Projects

Sustainable Peace

We have a professor on campus that needs help using Python to crunch data concerning "Sustainable Peace." This project uses matplotlib, numpy and scipy to create graphs. If you've any experience with Python and their math libraries, come join.

Flushing Food Map

The Flushing Chamber of Commerce asked if we can create buttons that apply filters to our existing data. Additionally, he would like to have the data be updated every day or week using the Yelp API. PHP and JavaScript are both used for this project.

Quic.NYC

Yup, that's us. We need to add a feature that saves student applications to our server/database. PHP is used for this project.

Building Temperature Maps

We want to use the data collected here to create GUI (map) that shows what rooms at CUNY John Jay are at the right temperature and which ones are too hot/cold. This information will be useful to building contractors.

Grayscale (Web/Android) App

This app would be used by the colorblind. Kevin built a working web demo here that needs some visual work and a 'save' button.

Additionally, there is interest in creating an Android app that would filter the screen (make it gray, block blue light) at the press of a button. Basically, something similar to this app.

Web Development Course

I am creating a curriculum for a web development course. Please view the page and help add to the curriculum.
